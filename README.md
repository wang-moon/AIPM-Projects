# AI产品经理项目作品集
欢迎访问我的 AI 产品实践仓库，这里整理了我从 0 到 1 设计并落地的两个完整 AI 项目：
智能客服 AI Agent & 企业培训机器人。

## 项目概览
### 01 企业级智能客服 AI Agent（电商场景）
覆盖订单查询、物流跟踪、退款售后、人工转接等全流程客服场景，通过意图体系、RAG 知识库、工具调用与风控规则实现高自动化客服解决方案。

核心产出：
- 需求文档 PRD
- 对话流程图
- 意图与槽位体系
- FAQ 知识库与 RAG 检索策略
- Prompt 工程与合规规则
- 工具调用接口规范
- Bad Case 复盘与数据报告

适用岗位：AI产品经理 / 电商客服产品经理 / 智能客服产品经理

---

### 02 企业内部智能培训机器人
面向新员工入职培训与在岗员工知识查询，提供知识点讲解、流程指引、答疑巩固与学习进度追踪。

核心产出：
- 培训知识体系
- 知识点结构化文档
- 交互流程图
- 产品 PRD
- 数据复盘报告

适用岗位：知识型 AI 产品经理 / 企业培训产品经理
