# AIPM-Projects
# AI产品经理项目作品集
本仓库包含2个完整的AI产品实战项目，覆盖**电商智能客服Agent**与**企业内部培训机器人**，还原AI产品经理从需求梳理、方案设计到落地复盘的全流程工作产出。

## 项目列表
### 01. 企业级智能客服AI Agent（电商场景）
面向电商售后全链路的智能客服解决方案，聚焦订单查询、物流跟踪、退款售后、人工转接等核心场景，实现咨询自动化与合规化运营。
核心产出：PRD、对话流程图、意图槽位体系、RAG知识库、Prompt工程、工具调用接口设计、风控规则、数据复盘报告等
适用场景：电商客服产品、AI Agent产品设计

### 02. 企业内部智能培训机器人
面向企业员工培训的对话式AI工具，解决新员工上手慢、老员工知识查询效率低的问题，实现知识自助答疑、流程讲解、知识点巩固。
核心产出：培训知识体系、产品PRD、交互流程图、知识点结构化文档、数据复盘
适用场景：企业培训产品、知识型AI产品设计
